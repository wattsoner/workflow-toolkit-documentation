Introduction

Unity Workflow Toolkit is a set of Plugins / Tools for Unity. This project focuses on delivering QOL (Quality Of Life) Features, making it easier for developers to create and manage their projects

About Workflow Toolkit

The Tools have a focus on improving Quality Of Life inside of the Unity Engine, it lets you squeeze a bit more out of time by automating boring and/or repetitive tasks

Technology

This toolkit mainly uses in-built Unity libaries, making it support a range of different versions and projects

Getting Started

This toolkit has been designed to be as easy to install and get started as possible, once purchased from the Unity Asset Store (or Itch.io) Import the .unitypackage file, and get started.

In-Engine help

There is a "Help" Section inside of the toolkit, please refer to that, or this documentation if necessary.

Important: Some tools use filepaths, it is extremely unlikely that the tools might delete the wrong file/folders, so please use tempoary ones
Use Help Window for more information.

Adding the Tools

Purchased from theUnity Asset Store:

To download the Workflow Toolkit from the Unity Asset Store:

  • Open the Unity Editor.
  • Go to the "Asset Store" tab or visit the Unity Asset Store website.
  • Search for "Unity Workflow Toolkit" in the search bar.
  • Click "Purchase" or "Download" if you've already purchased it.
  • Once the package is purchased, click the "Open in Unity" button, which will import the toolkit directly into your project.
  • After the package is imported, you can access it from your "Project" tab in Unity.

Purchased from Itch.io:

To download the Workflow Toolkit from Itch.io:

  • Visit the Itch.io website and log into your account.
  • Search for "Unity Workflow Toolkit" in the search bar.
  • Click "Purchase" or "Download" if you've already purchased it.
  • Once purchased, download the package to your local machine.
  • Unzip the downloaded file (if necessary) and manually import it into your Unity project by going to "Assets > Import Package > Custom Package" in Unity.
  • Browse to the location of the downloaded file, select it, and click "Open" to import the package.
