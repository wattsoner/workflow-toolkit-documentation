Introduction

Unity Workflow Toolkit is a set of Plugins / Tools for Unity. This project focuses on delivering QOL (Quality Of Life) Features, making it easier for developers to create and manage their projects.

About Workflow Toolkit

The Tools have a focus on improving Quality Of Life inside of the Unity Engine, letting you automate boring and/or repetitive tasks.

Technology

This toolkit mainly uses in-built Unity libraries, supporting a wide range of versions. Third-party libraries include NLog and Newtonsoft.

Getting Started

This toolkit has been designed to be as easy to install as possible. Once purchased from the Unity Asset Store (or Itch.io), import the .unitypackage file, and you're set.

In-Engine help

There is a "Help" Section inside of the toolkit; please refer to that, or this documentation if necessary.

Important: Some tools use file paths. Though unlikely, they could delete the wrong files/folders. Please use temporary directories if unsure. Use the Help Window for more information.

Adding the Tools

Purchased from the Unity Asset Store:

To download the Workflow Toolkit from the Unity Asset Store:

  • Open the Unity Editor.
  • Go to the "Asset Store" tab or visit the Unity Asset Store website.
  • Search for "Workflow Toolkit" in the search bar.
  • Click "Purchase" or "Download" if you've already purchased it.
  • Once purchased, click the "Open in Unity" button to import the toolkit directly into your project (or use Unity's "Package Manager").
  • After import, you can access it from the Toolbar or Windows menu.

Purchased from Itch.io:

To download the Workflow Toolkit from Itch.io:

  • Visit the Itch.io website and log into your account.
  • Search for "Workflow Toolkit" in the search bar.
  • Click "Purchase" or "Download" if you've already purchased it.
  • Once purchased, download the package to your local machine.
  • Unzip the downloaded file (if necessary) and manually import it into Unity via Assets > Import Package > Custom Package.
  • Select the file and click "Open" to import the package.
